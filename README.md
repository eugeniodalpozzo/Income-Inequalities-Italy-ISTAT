# Exploring Income Inequalities in Italy

This repository contains a research project analyzing income inequalities in Italy using microdata provided by ISTAT (Italian National Institute of Statistics).

The study examines the evolution and determinants of income inequality between 2015 and 2021, focusing on regional disparities, gender gaps, education, age, citizenship, and income sources.
